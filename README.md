# Storytelling com SQL — Analisando dados de RH

## Objetivo do Estudo
Essa é uma análise de RH de uma organização fictícia criada por cientistas de dados da IBM, em que foi utilizado apenas SQL para verificar os dados. O objetivo do trabalho é compreender o que leva os funcionários a saírem da empresa e como melhorar o ambiente de trabalho e a retenção de talentos.

Para verificar o artigo completo, pode acessar o link abaixo:
 - [Storytelling com SQL — Analisando dados de RH](https://medium.com/@henriquewfranco/storytelling-com-sql-analisando-dados-de-rh-ce60ec2f3122)

## Fonte dos Dados
Os dados foram retirados do [Kaggle](https://www.kaggle.com/), do dataset denominado [“IBM HR Analytics Employee Attrition & Performance”](https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset/data).
